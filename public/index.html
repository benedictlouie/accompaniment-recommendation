Accompaniment Generation

Groove Retrieval + Chord Snapping

🗄
LPD-5 Groove Index
957,530 bars from the Lakh Pianoroll Dataset, compressed to 16-dim onset vectors.
📏
Manhattan k-NN
Top-10 nearest bars sampled with temperature for varied groove texture.
🎼
Chord Snapping
Each retrieved note mapped to the nearest chord tone.
⚡
Real-Time Synthesis
Web Audio + MusyngKite soundfonts — piano, bass, guitar, drums. No server round-trips.
